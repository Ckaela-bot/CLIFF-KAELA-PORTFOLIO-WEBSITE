PORTFOLIO WEBSITE - QUICK START

1. OPEN THE WEBSITE
   - Open index.html in your browser
   - Or use VS Code Live Server extension (right-click index.html > Open with Live Server)

2. MAIN SECTIONS
   ✓ Hero/Home - Your impressive introduction
   ✓ About - Your background and skills
   ✓ Projects - Your major projects and achievements
   ✓ Achievements - Awards and recognition
   ✓ Resume - Qualifications and experience
   ✓ Contact - Ways to reach you

3. PERSONALIZATION
   Three easy files to edit:
   
   📄 index.html
   - Update your name, subtitle, descriptions
   - Modify project details
   - Change contact information
   
   🎨 styles.css
   - Change colors (look for :root section at top)
   - Adjust fonts and sizes
   - Modify animations
   
   ⚡ script.js
   - Add custom interactivity
   - Modify form behavior
   - Add tracking/analytics

4. KEY CUSTOMIZATIONS
   
   CHANGE COLORS:
   Open styles.css and find:
   --primary-color: #00d4ff;  (main cyan)
   --secondary-color: #0066cc; (blue)
   --accent-color: #ff6b35;    (orange)
   
   ADD RESUME PDF:
   Put your resume.pdf in the same folder as index.html
   Then find this line in index.html:
   <a href="mailto:kaelacliff425@gmail.com..." 
   
   Replace with:
   <a href="resume.pdf" download class="btn btn-primary">

5. ADD PROJECT IMAGES
   - Add your project images to the assets folder
   - Replace the icon (<i> tag) with:
     <img src="assets/project-name.jpg" alt="Description">
